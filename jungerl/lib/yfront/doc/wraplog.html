Description

wraplog is a simple wraplog. It attaches itself to the standard error logger so that all error messages produced by the error_logger module ends up in the wraplog.

At startup, a file is specified to the start/1 function. This file will get a number prefix for each wraplog. The number of wraplogs and the size of each file can be specified with the start/3 function. Per default, 9 files will be used, each having a size of 500 KB.

Function Details

start/1

start(LogFile) -> term()

Start the specified wraplog. Example: start("/tmp/mywraplog").

start/3

start(LogFile, LogSize, NoLogFiles) -> term()

Start the specified wraplog with specified size and number of files. Example: start("/tmp/mywraplog", 256000, 5). This will create a file: /tmp/mywraplog.1
