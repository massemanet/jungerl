Authors: Torbjörn Törnkvist (tobbe@tornkvist.org).

Description

Ymnesia is a Yaws appmod to view Mnesia tables. Add ymnesia as an appmod to your Yaws configuration. Point your browser to: <url>/ymnesia
  • You can search on arbitrary Erlang terms.
    (atoms need to be single-quoted)
  • The checkbox control if the attribute should be shown in the result.
    No checkbox means: show all attributes.

To test it, run the wblog example and put ymnesia in the URL path.
