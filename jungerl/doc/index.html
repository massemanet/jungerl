Jungerl

The Jungle of Erlang code.

What is in the collection?

See the jungerl ewiki page for possibly more up to date information.

Library
 Description
builder OTP release script builder.
ce miscellaneous Erlang/OTP programming support library
claw CLAW is an embryonic compiler from Core Erlang to Common Lisp.
edoc EDoc lets you write the documentation of an Erlang program as comments in the source code, using Javadoc-style tags and XHTML markup.
enfs Minimal NFS v2 server in Erlang
erl_cgi a cgi-erlang interface with a lot of nice features, such as esp srcipts
ermacs an erlang clone of emacs.
ex11 This is an Erlang-X11 binding, i.e an implementation of the X11 protocol in Erlang.
lersp an Erlang interpreter for an approximation of Scheme
msc Miscellaneous small (single-file) libraries.
pan Profiler
pico Joe Armstrong's pico (http server) and wiki code
rpc SUN ONC/RPC for erlang.
slang an erlang interface to the amazing highly portable tty interface slang
syntax_tools This package contains modules for handling abstract Erlang syntax trees, in a way that is compatible with the "parse trees" of the standard library module `erl_parse', together with utilities for reading source files in unusual ways and pretty-printing syntax trees. Also included is an amazing automatic code-cleaner.
tuntap Linux "Universal TUN/TAP device" driver
wiki Joe Armstrong's wiki code
wumpus Erlang Wumpus is an implementation of the classic game of _Hunt the Wumpus_
in Erlang.
www_tools This is Joe Armstrong's www_tools contrib
x11 New flavour of the ex11.
xmerl XML processing tools for Erlang
xmlrpc This is an HTTP 1.1 compliant XML-RPC library for Erlang.
measurement
 Measurement ADT
